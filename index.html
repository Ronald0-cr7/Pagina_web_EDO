Pizarra de series para ecuaciones diferenciales

Define tu ecuación de primer o segundo orden, con coeficientes polinómicos arbitrarios (no necesariamente 1 delante de y''). El programa detecta si x₀ = 0 es un punto ordinario o un punto singular regular, arma la relación de recurrencia (o la ecuación indicial y el método de Frobenius, según corresponda) y construye la solución general — siguiendo el capítulo de soluciones en serie para ecuaciones diferenciales lineales.

P₀(x)y'' + P₁(x)y' + P₂(x)y = G(x) -> clasificar x₀=0 -> (dividir entre P₀ | ecuación indicial) -> recurrencia -> solución general

Definir la ecuación

Los coeficientes se ingresan como polinomios en x, separados por comas, empezando en x⁰. Ej: "0,1" significa 0 + 1·x = x.

y' = A(x)·y + B(x)

P₀(x)·y'' + P₁(x)·y' + P₂(x)·y = G(x)

Si P₀(0) = 0, la herramienta trata x₀=0 como punto singular y aplica el método de Frobenius (no se piden y(0), y'(0): se muestra la familia de soluciones para x>0).

Se usa solo para comparar visualmente y calcular el error (solo aplica en puntos ordinarios). Déjalo vacío: la herramienta intenta reconocer la solución automáticamente.
